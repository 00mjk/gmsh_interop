Welcome to gmsh_interop's documentation!
========================================

This package allows Python to interoperate with the `gmsh <http://gmsh.info/>`_
mesh generator.

This package contains:

* :mod:`gmsh_interop.reader` to read gmsh's ``.msh`` file format.
* :mod:`gmsh_interop.runner` to run gmsh under program control and process its output.
